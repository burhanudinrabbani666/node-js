## Synatax Error

A syntax error is a mistake in code that breaks the rules (grammar) of a programming language, like missing a comma, parenthesis, or misspelling a command, preventing the code from running; it's like a grammatical error in human language, caught by compilers/interpreters before execution, often indicated by messages like invalid syntax pointing to the problem area
